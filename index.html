асинхронного двигателя На данном сайте вы сможете найти ссылку на ПО, позволяющее провести расчёты характеристик асинхронного двигателя в учебных целях.
Описание Программа написана на языке python в качестве курсового проекта по предмету объектно-ориентированное программирование. Приложение предназначено для облегчения расчётов студентов при проектирования асинхронного двигателя.

Для начала работы с приложением вам нужно перейти по ссылке и скачать архив. После скачивания разархивируйте любым удобным для вас способом архив и выберите приложение «Program.exe». Запустите приложение. Далее вас встретит стартовый экран с инструкцией по использования программы.
