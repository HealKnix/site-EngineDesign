Описание

Программа написана на языке python в качестве курсового проекта по предмету объектно-ориентированное программирование. Приложение предназначено для облегчения расчётов студентов при проектирования асинхронного двигателя.

Для начала работы с приложением вам нужно перейти по ссылке и скачать архив. После скачивания разархивируйте любым удобным для вас способом архив и выберите приложение «Program.exe». Запустите приложение. Далее вас встретит стартовый экран с инструкцией по использования программы.

Руководство

Раскройте более полную информацию, нажав на интересующие подсказки внизу.

1. Ввод данных

Введите данные согласно выданному варианту и нажмите "Ввести значения".

2. Заполнение таблиц

Введите полученные значения для вашего варианта в таблицу и нажмите кнопку "Проверка".

3. Проверка таблиц

При правильно введённых значениях вам откроется возможность ввода переменных для расчёта таблицы.

4. Ввод переменных

После ввода переменных нажмите кнопку "Рассчитать".

5. Расчёт таблицы

После нажатия кнопки "Рассчитать" происходит расчёт всей таблицы.

6. Графики

После нажатия кнопки "Графики" выскакивает окно с графиками.

7. Дальнейший расчёт

Расчёт последующих таблиц аналогичен.

В случае возникновения трудностей при подсчёте, перейти во вкладку "Настройки" и увеличьте значение погрешности.
